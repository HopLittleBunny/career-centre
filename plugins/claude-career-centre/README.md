# Career Centre for Claude

Career Centre is a skills-only Claude plugin by Amit Sharma. It turns a normal Claude conversation into an evidence-safe career workflow for role selection, Word CVs, cover letters, application tracking and interview follow-through.

Amit Sharma publishes the project through the GitHub account and repository owner `HopLittleBunny`.

## Install

### Public marketplace

1. In Claude, open **Customize > Plugins**.
2. Choose **Add marketplace**.
3. Enter `HopLittleBunny/career-centre` or `https://github.com/HopLittleBunny/career-centre`.
4. Install **Career Centre**.

### ZIP fallback

Download the Claude plugin ZIP from the [Career Centre install page](https://hoplittlebunny.github.io/career-centre/install.html), then upload it under **Customize > Plugins > Personal plugins**.

Start a normal chat and say: **Help me find my next role.** No Project, API key or separate Career Centre account is required.

## Boundaries

- Career Centre does not auto-submit applications or send external messages.
- It does not invent experience, metrics, qualifications, work rights or technical depth.
- Every displayed role needs an exact posting URL and salary/employment context.
- CV and cover-letter files are Word DOCX by default.
- The plugin has no Amit-operated backend or CV database. Claude processes files under Anthropic's own terms and plan limits.

Source, privacy, terms and support are available at [Career Centre](https://hoplittlebunny.github.io/career-centre/).
