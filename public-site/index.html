Simple defaults, with deeper control when you want it.

Say “change my advanced preferences” at any time. You never need to edit a configuration file.

Global by design

Market, currency, work-right rules, salary language and search sources follow the candidate. Australian assumptions do not leak into other countries.

CV fields

Show or hide contact fields, work rights, links and optional sections without changing the underlying evidence.

Page strategy

Experienced candidates default to two strong pages with page two at least 80% filled. Early-career candidates can use one strong page.

Section order

Change the section plan in ordinary language while retaining evidence checks and Word quality gates.

Reference CV formatting

Supply a Word CV as a visual model. Layout and styling can be reused, but the reference person's claims, links, metadata and private details are excluded.

Memory that stays visible and portable.

Preferences, corrections, role decisions and outcomes can be stored in a Career Passport inside the continuing conversation or exported to a local workspace.

Continuing conversation

The main career task remains the reliable place for application history, feedback and next actions.

Scheduled alerts

ChatGPT browser schedules use a saved snapshot. They suppress known roles and duplicates within a run, but later runs can repeat newly discovered roles when the host does not provide persistent state.

Honest boundaries are part of the product.

No invented evidenceEmployers, metrics, tools, qualifications and achievements cannot be added without support.
No generic Apply decisionAn exact, open posting and enough evidence are required before a role can be marked Apply.
No silent reference copyingA reference CV is a visual source only. Its content and private data are not candidate evidence.
No outcome guaranteesFit and shortlist estimates are decision support, not promises of interviews or offers.
