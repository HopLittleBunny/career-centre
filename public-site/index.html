Questions, answered

Before you install

It should feel like adding a capable career partner, not setting up software.

Is Career Centre really free?

There is no separate Career Centre subscription. You use your own ChatGPT or Claude plan, so provider plan and usage limits still apply.

Is automatic scheduling included on every plan?

No. Career Centre always offers the next step, but recurring Scheduled tasks require a provider plan and account that supports them. One-off searches still work without scheduling.

Why can the ChatGPT and Claude icons look different?

The ChatGPT package includes the Open Door directory and composer images. Claude's personal plugin ZIP currently shows its own plugin presentation and does not take a separate image asset from this package.

Does it remember everything forever?

Keep one main conversation and save the generated Career Passport. A new chat may not inherit earlier context, so re-upload the Passport when moving.

Will it apply to jobs automatically?

No. It finds, assesses and prepares. You make the decision and submit each application manually.

Can I use my own Word CV template?

Yes, when you ask. Career Centre can reuse the visual system of a permitted reference Word CV without copying that person’s content or metadata.
