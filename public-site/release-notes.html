Release notes

Functional beta. The approved beta.2 remains available while beta.3 packages the improvements found in the first real end-to-end application test.

Current build

4.0.0-beta.3

  • 60 automated tests pass in each provider package.
  • A broad CV-strength verdict and optional deeper review now appear during intake.
  • The first Career Passport is prepared automatically after setup.
  • Daily or weekday scheduling is offered after the first successful manual search.
  • ChatGPT Work schedules prefer the existing task and its context; standalone fallback remains explicit.
  • India, United States and Australia localisation rules now cover salary language, work rights, document conventions and market sources.
  • Fresh fictional India and US setup/search journeys pass, including Naukri-aware source selection, exact posting checks and honest no-filler results.
  • Compensation floors now retain their basis and use local display, including INR lakh formatting.
  • Every generated CV receives a transparent quality rating and revision threshold, separate from ATS or shortlist scoring.
  • A reference Word template is supported only when the user explicitly requests it.
  • The listing now explains that Career Centre has no separate subscription and uses the user's provider plan.
  • Natural CV-first setup and a seven-line readiness receipt pass.
  • Multi-CV intake, cross-version source mapping and qualitative first CV review pass.
  • The Career Passport now serves as the portable Career Evidence File and history backup.
  • Generated CV and cover-letter versions retain source provenance and ready, partial or superseded status.
  • A transparent CV diagnostic gives impact-ranked parseability and writing findings without a pretend universal ATS score.
  • Optional language and regional-spelling preferences remain hidden until requested.
  • Full CV content is returned in chat as a partial fallback when Word creation is unavailable.
  • Global market isolation and advanced CV preferences pass.
  • Paired Word CV and cover-letter generation passes rendered QA.
  • Natural reference-CV formatting passes evidence and metadata isolation.
  • Standalone scheduled alerts disclose snapshot-backed continuity and repeat risk.
  • Claude custom-plugin upload, normal-chat auto-routing, mentor reflection and readiness receipt pass.
  • A GitHub-backed Claude marketplace provides a simpler install route.
  • No automatic application submission or external messaging.

Known release gates

The approved OpenAI beta.2 and submitted Claude build remain the public versions until beta.3 is submitted as an update. ChatGPT Work beta.3, broader-plan compatibility and a two-run Claude Cowork continuity test remain open.
