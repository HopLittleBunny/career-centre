Upload it

  1. Open ChatGPT on the web.
  2. Open Profile, Skills, Create, Upload from your computer.
  3. Select the downloaded ZIP and allow ChatGPT to scan it.
  4. Open a normal new conversation and say Help me find my next role.
  5. Share your CV when asked.
No Project is required. There is no API key, publisher account, manual tracker setup or prompt syntax.

What happens first

The skill reads the CV, asks only for missing decisions that could change the result, and then shows Your Career Centre is ready with the active market, sources, salary, employment and CV assumptions.

If upload is unavailable

Personal skill upload can depend on the user's plan, region and current ChatGPT surface. Try ChatGPT web and check the Plugins and Skills directory. Do not create a Project as a workaround.

Directory installation

After OpenAI review, the public listing will offer the simpler directory route. Users will find Career Centre, select Install and start a normal conversation.

Document safety

Never upload another person's CV or reference document unless you have permission to use it. Reference CVs are treated as visual models only.

Free to use with the user's ChatGPT plan limits.
