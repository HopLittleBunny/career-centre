# Install in ChatGPT

## Direct Skill ZIP

1. Download the latest official Career Centre Skill ZIP.
2. Open **Profile → Skills → Create → Upload from your computer** in ChatGPT.
3. Select the ZIP and let ChatGPT scan it.
4. Start a normal new chat and say **Help me find my next role.**
5. Share your CV when asked.

No Project, API key, tracker setup, publisher account or separate chat structure is required. The skill uses your ChatGPT plan and its normal limits.

If the Skills page or upload control is unavailable, your plan, region or current ChatGPT surface may not support personal skill upload yet. Try ChatGPT web and check the Plugins/Skills directory. Do not create a Project as a workaround.

## Directory installation

After public review, Career Centre will also have a Plugin-directory listing. That route will reduce setup to finding the listing, selecting **Install**, and starting a normal chat.

## First conversation

The skill reads your CV before asking questions. It then asks only for missing decisions that could change the result and shows **Your Career Centre is ready** with the active market, sources, salary, employment and CV assumptions. You can later say **change my advanced preferences** if you want different sources, sections, page strategy or reference-CV formatting.

Never upload another person's CV or reference document unless you have permission to use it.
