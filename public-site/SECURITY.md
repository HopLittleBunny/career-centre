# Security and private-data reports

Do not include CV content, personal contact details, identity documents, work-right evidence, private employer material, account cookies, tokens or login information in a public GitHub issue.

For ordinary product problems, use the repository issue form with redacted information only. If a report cannot be safely redacted, do not publish it in the issue tracker.

Career Centre has no publisher-operated CV database, authentication system or application-submission service. User conversations, attachments and generated files remain within the host environment and the user's own storage choices.
