# Initial submission release notes

Career Centre is a skills-only career decision plugin by Amit Sharma. A user starts in ordinary language, shares one CV or several role-track-specific CVs, and receives a constructive first CV review, selective role research, evidence-led fit commentary, exact posting links with salary/employment context, paired Word application packs and application follow-through. The Career Passport now acts as a portable, source-mapped Career Evidence File and history backup. If Word creation is unavailable, the complete CV is returned in chat as a clearly labelled partial fallback.

This is the initial public submission. It has no MCP server, external authentication, publisher-operated database or demo credentials. Review fixtures are synthetic. The plugin never auto-submits applications and treats CV-derived claims as candidate-provided, source-only evidence.

Release evidence includes 56 contract, document and fail-closed integrity tests in each provider package plus rendered Word QA. The beta.2 enrichment adds a transparent non-scored CV diagnostic, generated-document version history and optional language/regional preferences without a third-party runtime dependency. The ChatGPT package has passed installed browser journeys; the preceding Claude package passed ZIP upload, normal-chat routing, mentor reflection, the global readiness receipt and an explicit setup-only/no-search boundary. The final beta.2 Claude ZIP must pass replacement upload and smoke testing before submission.
