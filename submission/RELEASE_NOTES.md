# Beta.4 update release notes

Career Centre now treats the daily-search handoff as a required product milestone. The first verified manual search ends with a visually separate offer to make the calibrated search automatic, even when every role is Maybe or Skip. ChatGPT Work can create the Scheduled task after the user supplies cadence, time and timezone. Claude uses Cowork's `/schedule` flow. If the first offer is missed or ignored, the first completed application pack repeats it once as a recovery path.

The update also includes the warm Open Door identity in the package. During the OpenAI update, the Open Door asset must be uploaded explicitly to every directory and composer icon slot; packaged assets alone do not replace an existing listing icon.

Career Centre is a skills-only career decision plugin by Amit Sharma. A user starts in ordinary language, shares one CV or several role-track-specific CVs, and receives a constructive first CV review, selective role research, evidence-led fit commentary, exact posting links with salary/employment context, paired Word application packs and application follow-through. The Career Passport now acts as a portable, source-mapped Career Evidence File and history backup. If Word creation is unavailable, the complete CV is returned in chat as a clearly labelled partial fallback.

This is the initial public submission. It has no MCP server, external authentication, publisher-operated database or demo credentials. Review fixtures are synthetic. The plugin never auto-submits applications and treats CV-derived claims as candidate-provided, source-only evidence.

Release evidence includes contract, document, fail-closed and milestone-specific automation tests in each provider package plus rendered Word QA. Beta.4 must also pass a live ChatGPT Work schedule-creation smoke test and a Claude Cowork `/schedule` handoff test before the automation defect is considered closed.
