# Initial submission release notes

Career Centre is a skills-only career decision plugin by Amit Sharma. A user starts in ordinary language, shares a CV, and receives selective role research, evidence-led fit commentary, exact posting links with salary/employment context, paired Word application packs and application follow-through.

This is the initial public submission. It has no MCP server, external authentication, publisher-operated database or demo credentials. Review fixtures are synthetic. The plugin never auto-submits applications and treats CV-derived claims as candidate-provided, source-only evidence.

Local release evidence includes contract, document and fail-closed integrity tests plus rendered Word QA. The final submission ZIP must be produced from the same plugin tree used in live ChatGPT testing.
