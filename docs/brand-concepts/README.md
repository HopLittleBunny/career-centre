# Career Centre logo directions

These are selection concepts, not live assets. Each is a scalable 512 × 512 SVG designed to remain legible at plugin-icon size.

## A — Sunrise Path

The most career-specific option. A warm open `C` holds a rising path and sun: direction, progress and optimism without looking like a recruitment agency.

## B — Open Door

The warmest and most welcoming option. An open arch and path suggest a trusted place where the next opportunity becomes clearer. It is less literal about the letter `C`.

## C — Conversation Spark

The best expression of the mentor/friend promise. Two rounded conversation forms create a subtle `C` around a small spark, signalling thoughtful dialogue and forward movement.

Recommendation: test A and C at 48 px in ChatGPT/Claude dark mode before choosing. B is strongest emotionally but needs the product name beside it for instant category recognition.
