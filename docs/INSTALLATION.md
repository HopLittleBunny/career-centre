# Install Career Centre

## Fastest pre-release route: ChatGPT Skill ZIP

Where the Skills page is available:

1. Download the `career-centre-…-skill.zip` release file.
2. In ChatGPT, open **Profile → Skills → Create → Upload from your computer**.
3. Select the ZIP and accept it only after ChatGPT's safety scan passes.
4. Turn the skill on if ChatGPT does not enable it automatically.
5. Start a normal new chat and say: **Help me find my next role.**
6. Share a CV when asked. Word, PDF, LinkedIn export or pasted text works.

That is the entire user setup. There is no project, repository, tracker, API key, publisher account or MCP server.

The first useful flow should be:

- the skill reads the CV;
- it reflects back what it understood;
- it asks one compact set of only the decisions that are still missing;
- it shows **Your Career Centre is ready** with the active assumptions;
- it starts the requested search or role assessment without asking the user to configure internal settings.

If **Skills** or **Upload from your computer** is not present, availability may not yet cover that plan, region or surface. Use ChatGPT web first and check the account's Plugins/Skills page. Do not ask the user to create a Project as a workaround.

## Public-release route

After OpenAI review, the intended path is:

1. Open the ChatGPT Plugin directory.
2. Search for **Career Centre** or use Amit's direct listing link.
3. Select **Install**.
4. Start a normal chat with **Help me find my next role.**

The public listing route removes the ZIP step. It still uses the user's ChatGPT plan and limits.

## Updating

Until the public directory provides automatic updates, install a newer version only from Amit's official release page. Review the release notes, then replace or update the existing skill through the Skills page. Keep a downloaded Career Passport if you are moving to a new task or account.

## Privacy check before use

- The skill package contains instructions, templates and local document scripts.
- It has no Amit-operated CV database or external authentication.
- CVs and generated documents are still processed and stored by ChatGPT according to the user's OpenAI account settings and policies.
- Do not upload a CV or reference document you are not authorised to use.
