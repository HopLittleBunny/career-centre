# ChatGPT friend beta plan

## Current status

- Career Centre `4.0.0-beta.2` is **Approved**.
- It is **not yet Published**, which is why it does not appear in ChatGPT plugin search.
- The approved version's row menu in the OpenAI submission portal contains the separate `Publish` action.

## Recommended launch sequence

### 1. Quietly publish beta.2

Publish the already-approved version without announcing it broadly. Publication makes it available through the universal plugin directory and creates a directory page/link that can be sent directly to testers.

### 2. Share the exact directory link with five testers

Ask for a mix of:

- one early-career user;
- two mid-career users;
- one senior/executive user;
- one user with two or more distinct CV versions.

Do not rely on directory search for the beta. Send the exact Career Centre directory URL generated after publication.

### 3. Give testers this installation note

> Open the Career Centre link in ChatGPT, select **Install plugin**, switch to **Work**, start a new task and say: **“Help me find my next role.”** Share your latest CV when asked; if you use different CVs for different role directions, share the key versions together.

### 4. Run five short journeys

1. **First-use clarity** — Does it ask only for a CV first and explain what happens next?
2. **Multi-CV intake** — Does it keep versions and claims separate and explain which base suits each direction?
3. **Role assessment** — Does every role include an exact link, salary context, honest fit, risk and Apply/Maybe/Skip recommendation?
4. **Document output** — Does it create a polished two-page Word CV and cover letter, or provide the complete CV in chat if Word creation fails?
5. **Continuity** — Does it update the Career Passport and clearly explain how to continue in one main task or move to a new one?

### 5. Collect comparable feedback

Ask every tester:

1. Was installation obvious?
2. Did the first response feel like a career mentor or a form-filling bot?
3. Did it ask too many questions?
4. Were role recommendations selective and credible?
5. Did any CV claim feel invented or overstated?
6. Was the Word output genuinely usable?
7. What was the single most confusing moment?

## Success threshold

The beta is ready for a wider launch when at least four of five testers can install it without help, complete the intended journey, understand its memory limits, and receive either a usable Word document or a complete in-chat fallback without fabricated evidence.

## External-friend versus workspace sharing

- **Friends in different ChatGPT accounts/workspaces:** publish the plugin and send its exact directory URL.
- **Members of the same ChatGPT workspace:** the desktop app can share a plugin from `Plugins → Created by you → Share`; recipients see it under `Shared with you`.
- Workspace sharing is not an unlisted public beta. People outside that workspace cannot use the shared item.

## Beta.3 immediately after publication

Create a new draft version on the same plugin with:

- the new compass-C icon in light and dark variants;
- three legible marketplace story panels;
- any fixes found in friend testing;
- concise release notes.

Submit beta.3 for review while beta.2 remains available. When beta.3 is approved, publish the update to replace beta.2 without creating a second Career Centre listing.
